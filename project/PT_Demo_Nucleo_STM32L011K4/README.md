# PT_Demo_Nucleo_STM32L011K4
   --       |  Description                              
 ---------- | ------------------------------
 Name       | PT_Demo_Nucleo_STM32L011K4                                         
 Function   | Demo application of 2 protothread tasks: Task 1 controls the Green LED blinking; Task 2 print task infomation periodically

 MCU        | STM32L011K4
 Type       | ARM Cortex-M0+
 Frequency  | 32MHz
 RAM        | 2K bytes
 ROM        | 16K bytes
 peripheral | LED * 1
  --        | Reset button * 1
