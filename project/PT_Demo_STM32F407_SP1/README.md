# PT_Demo_STM32F407
  --        |  Description                              
 ---------- | ------------------------------
 Name       | PT_Demo_STM32F407                                       
 Function   | Demo application of 2 protothread tasks: Task 1 controls the 2 LEDS blinking; Task 2 prints task infomation periodically                           
 Platform   | HengBo_DEV_BOARD_STM32F407                        
 MCU        | STM32F407
 Type       | ARM Cortex-M4
 Frequency  | 168MHz
 RAM        | 192K bytes
 ROM        | 512K bytes
 peripheral | LED * 2
  --        | Reset button * 1
  --        | User button * 2
